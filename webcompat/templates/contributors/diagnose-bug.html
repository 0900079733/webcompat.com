• Find a bug that "Needs Diagnosis"
  • Check to see if bug has been replicated already
  • If not, try to replicate the bug
  • If bug is NOT reproducible, write a comment on the issue
  • If bug is reproducible, try to reproduce on other browsers/OS
  • Check the console to see if there are errors
  • Google any errors
  • Write a comment with your findings on the issue
  • If you find a fix, post your fix on the issue!
  • If you not, post what you tried on the issue
