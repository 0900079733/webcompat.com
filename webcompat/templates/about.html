About Webcompat.com

What is webcompat.com?

Webcompat.com was developed by Mozilla to open the web compatibility effort to all users of the web. It is a single place for users to log web compatibility bugs for any browser. Developers can use it to get meaningful feedback about compatibility issues with their sites.

What is Web Compatibility?

Web compatibility is about ensuring that web users everywhere have a choice of which web browser or device they would like to use. This happens through finding compatibility issues on websites, understanding the issue, contacting the website and helping to resolve. The great thing is anyone can help, learn how you can get involved!

Interested in learning more? Check out the Mozilla Compatibility Wiki.
